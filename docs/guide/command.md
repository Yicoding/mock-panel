# 命令说明

```bash
# 启动本地调试
yarn dev

# 启动 mock 数据
yarn mock

# 本地开发时使用测试环境配置
yarn start-test

# 使用本地环境配置构建
yarn build-dev

# 使用测试环境配置构建
yarn build-test

# 使用 uat 环境配置构建
yarn build-uat

# 使用正式环境配置构建
yarn build-prod

# 构建产物分析
yarn build-analyze

# 本地预览生产构建产物
yarn preview

#  git 提交时，commitlint 校验 commit message
yarn commit

# 自动提升版本，并生成日志
yarn changelog
```
